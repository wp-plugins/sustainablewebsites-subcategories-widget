== Description ==

This WordPress Subcategories Widget displays all subcategories on a WordPress category archive page.

The widget will not show un-necessarily on pages or posts.

This widget is a useful navigation feature that will help your web site visitors drill down into your category hierarchy to find the right category. It is completely style-able with CSS.

It's a simple widget that does exactly what it says! Drag to your sidebar to activate.

If you have any suggestions please email wordpress (at) sustainablewebsites (dot) com

== Installation ==

1. Extract the zipfile
2. Upload the folder to /wp-content/plugins
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag the widget to a sidebar in the Appearance | Widgets menu
