== Description ==

The [Sustainable Websites](http://www.sustainablewebsites.com) WordPress Subcategories Widget displays all subcategories on a WordPress category archive page.

The widget will not show un-necessarily on pages or posts.

This widget is a useful navigation feature that will help your web site visitors drill down into your category hierarchy to find the right category. It is completely style-able with CSS. It's also good for search engine optimization (SEO) to help your site's link structure be easily crawl-able.

It's a simple widget that does exactly what it says! Drag to your sidebar to activate.

Plugin homepage: [WordPress Subcategories Widget](http://www.sustainablewebsites.com/wordpress-subcategories-widget)

If you have any suggestions please email wordpress (at) sustainablewebsites (dot) com

== Installation ==

1. Use the Add New button in your WordPress Dashboard and search for Sustainable Websites Subcategories Widget

or 

1. Download and extract the zip file
2. Upload the folder to /wp-content/plugins
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag the widget to a sidebar in the Appearance | Widgets menu
